Trip Planner Project: Instruction

1. You should meet system requirement.
	- Linux-based web hosting server
	- PHP Version 5.6.30
	- MySQL Version 5.5.54 (Must support MySQLi Extension)
	- phpMyAdmin
	- FTP
	- SMTP for PHP main() function
	- Linux Permission: 755 (Directory), 644 (File)
2. Import "db.sql" file to your MySQL server
2. Open "mysqli.php" file to modify your server information
3. Upload all files
4. Set Linux permission: 755 (Directory), 644 (File)
5. Execute index.html
